# Task 2 - Credit Card Fraud Detection 

##  Objective
To build a machine learning model that detects **fraudulent credit card transactions** and classifies them as **legit or fraud**.

##  Dataset
Used the [Credit Card Fraud Detection Dataset](https://www.kaggle.com/datasets/kartik2112/fraud-detection).  
Contains **1.29M transactions** with **7.5K fraud cases** (`is_fraud` column).

##  Approach
1. **Loaded** and cleaned dataset using Pandas  
2. **Encoded** categorical data & scaled features  
3. **Trained** a **Random Forest Classifier**  
4. **Evaluated** using accuracy, confusion matrix & classification report  

##  Results
- **Model Used:** Random Forest  
- **Accuracy:** ~99.9%  
- **Precision (Fraud):** ~0.95  
- **Recall (Fraud):** ~0.92  

##  Technologies Used
- Python  
- Pandas, NumPy  
- Scikit-learn
